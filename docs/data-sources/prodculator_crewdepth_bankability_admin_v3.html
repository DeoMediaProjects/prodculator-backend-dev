How to read this
Bankable = how long it actually takes to get paid, government-sourced and cross-checked against real-world evidence — NOT a derived score. Ratings are PROVISIONAL: the bands (Most Bankable / Bankable / Slow / Not Bankable) are placeholder cutoffs from only 6 fully-verified territories. Do not treat these cutoffs as final. South Africa is the reference case for "Not Bankable" — confirmed suspended, confirmed real backlog, not just a stated policy read at face value.
